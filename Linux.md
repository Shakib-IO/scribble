### Commands

- How to determine Linux kernel architecture?
```
uname -a
# OR
uname -m && cat /etc/*release
# OR 
uname --help
````
- Search the available Drivers.
```
ubuntu-drivers devices
# OR
apt search nvidia-driver
```
